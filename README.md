# pdagames.github.io

Структура проекта для сайта разработчика мобильных игр:

- `assets/` — статические ресурсы.
  - `css/` — стили.
  - `js/` — скрипты.
  - `img/` — изображения.
  - `fonts/` — шрифты.
- `components/` — переиспользуемые фрагменты интерфейса.
- `pages/` — отдельные страницы сайта.
- `data/` — данные/контент (например, JSON с портфолио).

`index.html` остаётся точкой входа, а остальной контент размещается в папках выше.
